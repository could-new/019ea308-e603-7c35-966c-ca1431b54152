# El Estado Argentino - Atributos

Una aplicación educativa interactiva desarrollada en Flutter que explica en detalle los siete atributos fundamentales del Estado Nacional Argentino, acompañados de definiciones claras y ejemplos prácticos.

## Características

*   **Interfaz Responsiva:** Diseño adaptativo que funciona a la perfección en dispositivos móviles (vista en lista) y plataformas de escritorio o web (vista en cuadrícula).
*   **Contenido Educativo:** Cobertura de los conceptos fundamentales de formación cívica y derecho político:
    *   Territorio
    *   Población
    *   Uso de la fuerza
    *   Monopolio fiscal
    *   Organización administrativa
    *   Ordenamiento normativo
    *   Soberanía
*   **Ejemplos Prácticos:** Cada concepto abstracto está anclado a un ejemplo real de la República Argentina (ej: AFIP/ARCA, BCRA, Censo 2022, Fuerzas de Seguridad).
*   **Diseño Material 3:** Uso de paleta de colores inspirada en la bandera argentina con una estética moderna, clara y accesible.

## Ejecución Local

1. Asegúrate de tener Flutter instalado y configurado en tu sistema.
2. Clona el repositorio.
3. Instala las dependencias:
   ```bash
   flutter pub get
   ```
4. Ejecuta la aplicación:
   ```bash
   flutter run
   ```
